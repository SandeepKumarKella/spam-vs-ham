# Spam vs Ham Real-Time Email Classifier

This project connects to Gmail inbox and detects whether emails are spam or ham using Machine Learning.

## Features
- Real-time Gmail email reading
- Spam detection
- Naive Bayes classifier
- Python virtual environment
- GitHub integration

## Technologies
- Python
- Scikit-learn
- IMAP
- Pandas
